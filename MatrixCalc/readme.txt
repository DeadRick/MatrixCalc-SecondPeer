Инструкция по программе MatrixCalc.

1. Считавание матрицы по файлу.
	1.1. Путь к файлу.
		Для того, чтобы считать матрицу из файла, нужно верно указать путь.
		В коде изменить строку 62 string path = @"<Нужный путь>";
		Пример пути: C:\Users\Пользователь\source\repos\MatrixCalc\test.txt
		Если путь неверный, то Вам будет выкидвать ошибку.
	1.2 Заполнение файла.
		Вид матрицы в файле должен быть n x n, то есть квадратная матрица.
		Вводить элементы через пробел. Пример строки "1 2 3 4 5".
		После последнего элемента пробел не должен стоять!
		ВНИМАНИЕ!!! Вводить можно только числа! Кол-во строк должны быть одного размера.
		Если Вы ввели 5 цифр, то во всех следующих строчках должно быть РОВНО 5 цифр!
		В случае, если в файле не хватает строки, то строка автоматически заполнится нулями.
2. Заполнение матриц элементами.
	2.1 Рандомная матрица.
		Этот метод доступен для одной и двух матриц.
		После того, как Вы ввели размеры матрицы, нажмите "1" на клавиатуре,
		чтобы значения ввелись автоматически от -100 до 100! Такой интервал выбран
		для наглядности.
		ВНИМАНИЕ!!! Если Вы выбрали две матрицы и хотите их заполнить рандомными элементами,
		то Вы заполняете сразу ДВЕ матрицы!
	2.2 Заполнение вручную.
		Этот метод доступен для одной и двух матриц.
		После того, как Вы ввели размеры матрицы, нажмите "2" на клавиатуре,
		для того, чтобы ввести все значения вручную. Элементы можно вводить с n кол-вом нулей.
		Элементы можно вводить со знаком минус и плюс. Элементы доступны с -10000 до 10000.
	2.3 Считывание с файла.
		Этот метод доступен лишь для ОДНОЙ матрицы!
		Для того, чтобы запустить этот метод нужно выбрать 1 матрицу. Ввести 1 строку и 1 столбец,
		дальше нажать "3" на клавиатуре. Если выводится ошибка то, пункт 1.1.
3. Операции с матрицами.
	3.1 Одна матрица.
		3.1.1 Квадратная матрица.
			(1) Транспонирование матрицы.
			(2) Умножение матрицы на число.
			(3) След матрицы.
			(4) Определитель Матрицы.
		3.1.2 Не квадратная матрица.
			(1) Транспонирование матрицы.
			(2) Умножение матрицы на число.
	3.2 Две матрицы.
		3.2.1 Две квадратные матрицы.
			(1) Сложить матрицы.
			(2) Вычесть матрицы.
			(3) Умножить матрицы.
		3.2.2 Кол-во столбцы первой матрицы равны кол-ву строкам второй матрицы.
			(1) Умножить матрицы.
		3.2.3 Ни кол-во строков, ни кол-во столбцов не совпадает.
			(-) Операций нет.
